# Titanic Survival Data Analysis

## Project Overview
This project explores the famous Titanic dataset to understand the factors that influenced passenger survival during the 1912 maritime disaster.

The goal is to perform **exploratory data analysis (EDA)**, clean the dataset, and identify patterns in survival outcomes using Python.

---

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Git & GitHub

---

## Dataset Information
The dataset contains information about 891 Titanic passengers including:

- Age
- Gender
- Passenger Class
- Fare
- Family relationships
- Survival status

---

## Data Cleaning Performed
- Removed `Cabin` due to excessive missing values
- Filled missing `Age` values using median
- Filled missing `Embarked` values using mode
- Created new feature: `FamilySize`
- Created feature: `FamilyCategory`

---

## Key Insights

- Females had significantly higher survival rates than males
- First-class passengers had the highest survival probability
- Children had the highest survival rate among age groups
- Passengers traveling in small families (2–4 members) had better survival chances than solo or large groups

## Project Outcome
This analysis reveals that survival was strongly influenced by:
- Gender
- Passenger class
- Age group
- Family size

These insights demonstrate how real-world data can be used to understand historical events.
